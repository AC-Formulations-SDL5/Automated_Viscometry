# Results directory

CSV outputs, archived run data, and experiment exports live here.

- **New instrument runs:** `results/runs/YYYY-MM-DD/`
- **Archived root-level CSVs:** `results/runs/archive/`
- **Legacy auto-run exports:** `results/auto_runs/` and `results/Auto-runs/`

Analysis notebooks are maintained under [`notebooks/`](../notebooks/) at the repo root (not in this folder).
